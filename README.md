# Bins

```bash
cp .env.default .env
docker-compose up
```

```bash
# make a http request to a bin
curl $(docker-machine ip default):8080/bins/abc?foo=bar

# fetch the requests
curl $(docker-machine ip default):8080/v1/bins/abc
```
